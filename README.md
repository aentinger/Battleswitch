## Build
* Install AVR-GCC toolchain and cmake
```
sudo apt-get install cmake avr-libc binutils-avr gcc-avr avrdude
```
* Clone repository
```
git clone https://github.com/lxrobotics/Battleswitch.git
```
* Build
```
cd software
cmake .
make
```
